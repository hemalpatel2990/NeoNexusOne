Modular Level Kit — The 5 Pieces

Each piece is a Blueprint Actor with per-instance color. All use M_EchoMaster via Dynamic Material Instance.

  ┌────────────────────┐
  │                    │  400w × 50d × 400h
  │    WALL SEGMENT    │  Snap-friendly grid size
  │                    │  Box collision
  └────────────────────┘

BP_LevelWall

Mesh: Scaled Cube (400×50×400)

Use: Corridors, rooms, boundaries. The core structural piece.

        /\
       /  \
      /    \       300h cone
     /      \      150 base radius
    /________\     Box/convex collision

BP_LevelRock

Mesh: Cone (R:150, H:300)

Use: Mountains, rocky outcrops. Scale up for peaks, cluster small ones for rocky terrain.

       ___
      (   )        Sphere R:80 (canopy)
      (___)
        |
        |          Cylinder R:25, H:200 (trunk)
        |
      __|__

BP_LevelTree

Meshes: Cylinder trunk + Sphere canopy (compound)

Use: Forest areas. The sonar sweeping over round canopies creates beautiful reveals.

      ┌──────────┐
      │          │    100 × 100 × 100
      │  CRATE   │    Box collision
      │          │    Stackable
      └──────────┘

BP_LevelCrate

Mesh: Cube (100×100×100)

Use: Scattered obstacles, cover, stackable platforms. Works in any environment.

           /|
          / |
         /  |       200w × 200d × 150h
        /   |       Wedge shape
       /____|       Walkable surface

BP_LevelRamp

Mesh: UE5 Wedge or rotated Cube with collision (200×200×150)

Use: Elevation changes, hillsides. Connect flat areas to raised platforms.
